# 自动化测试方案

## 测试目标
验证宠物网站的关键闭环是否可用，而不是只验证接口返回 200。

## 工具
- pytest：运行测试
- Playwright：执行浏览器自动化
- Chromium：测试浏览器

## 覆盖范围
### 核心 E2E
- 首页可访问
- 用户注册
- 用户登录
- 创建宠物
- 喂食
- 玩耍
- 开启托管
- 创建或加入圈子
- 发起一次互动
- 查看事件日志或状态变化

### 后端基础测试
- 健康检查路由
- 宠物状态推进服务
- 托管策略逻辑
- 圈子互动冷却逻辑

## 运行方式
```bash
pytest -q
pytest tests/e2e -q
