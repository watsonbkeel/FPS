# DECISIONS

## D001 - 公开定位只保留方块 FPS

本仓库当前对外文档与 GitHub 首页均以 **方块 FPS 独立站** 为唯一公开定位。

## D002 - 独立站采用昵称会话

玩家只需输入昵称即可进入游戏，不依赖账号体系。

## D003 - 独立站独立部署

对外入口以 `app.fps_main` 为准，默认端口 `18428`。

## D004 - 结算只展示战斗结果与评分

当前独立站不绑定外部经济系统，结算以战斗结果、击杀、MVP、个人评分为主。

## D005 - 联机房间暂时保持轻量实现

当前房间服务为内存实现，优先保证单实例可玩和开发效率，后续再评估持久化与多实例方案。
